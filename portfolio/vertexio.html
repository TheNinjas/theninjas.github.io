Vertexio Typeface

Simple, All-Caps, Screen-Friendly Sans-Serif font.

The first typography project which I took up, Vertexio Font is more of an exploratory project with which I learnt how to design and create a font.Vertexio is an all-caps font, with strong and bold features which can be very well used as an heading font. It is inspired by modern swiss typefaces. It is freely available and can be downloaded from here
